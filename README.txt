This plugin is capable to edit contact forms created on Contact 7 Form to insert multiple recipients with pipes.

== Description ==

My goal to wrote this plugins is be able to easly create Multiple Recipients with pipes.

This plugin can create new recipients and implements in the form the necessary tags.

In this version it can't change Mail 2 options.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
